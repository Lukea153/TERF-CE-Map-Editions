# TERF-CE-Map-Editions
Adds both the CRR and TOASTER to the PMC version of TERF. As well as updating the map to the current version as seen on the server.

Now you can toast toast and play with weird redstone reactor stuff!

Requires both CRR and TOASTER, obv.


HOW TO SETUP:
After installing the datapack, type this command
/function ce_map_additions:setup_map/paste_terfmap

There should be a lagspike and after a couple seconds, the map will reset. It will now be the updated version of the map.
